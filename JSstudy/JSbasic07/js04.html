window 메서드

- 브라우져 객체의 최상위 객체이다.
- window 객체에 존제하는 메서드
- window를 생략하고 쓰일 수 있다.

open 메서드

- 새창 혹은 팝업을 띄울때 쓰인다. - 구문 : window.open('띄울 주소', '팝업 제목', '옵션');

alert 메서드

- 사용자에게 경고창 보여주기

prompt 메서드

- 사용자에게 값을 받는다.(질문에 의한 데이타를 사용자가 입력가능);

confirm 메서드

- 사용자에게 진행여부를 확인 받는다. - 사용자가 진행여부에 따라 결과값은 true 혹은 false로 전달된다.

close 메서드

- 사용자가 보고 있는 창을 닫는다.

setInterval 메서드

- 일정한 간격으로 인수값에 들어간 시간간격으로 함수를 실행

clearInterval 메서드

- setInterval 메서드 실행으로 실행중인 이벤트를 중지하는 메서드

setTimeout 메서드

- 인수값에 들어간 시간 후에 함수 실행

clearTimeout 메서드

- 인수값에 들어간 시간 후에 실행되기전에 취소 할때 쓰이는 메서드

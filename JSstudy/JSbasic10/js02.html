테그 속성 이벤트

- 모든 테그에는 on이라는 속성을 가지고 있다.
- on의 속성 명칭은 사용자의 행위에 관련한 명칭으로 되어있다.
- 함수와 연계하여 이벤트 처리를 한다.
- 각 이벤트 속성은 상황에 따른 테그에 부합하여야 한다.
- on 속성의 값은 자바스크립트 코드가 들어간다.

자주 쓰이는 이벤트 목록

- onclick : 사용자가 클릭 했을 경우
- onchange : 사용자가 데이타를 변화 시켰을 경우
- onblur : 사용자가 데이타 입력하다가 포커스를 빠져 나왔을 경우
- onfocus : 사용자가 데이타 입력을 하기위하여 포커스를 잡았을 경우(커서깜빡임)
- onkeyup : 사용자가 키보드를 눌렀다가 땠을 경우
- onkeydown : 사용자가 키보드를 눌렀을 경우
- onkeypress : 사용자가 키보드를 누르고 있을 경우
- onsubmit : 사용자가 문서양식을 전달 할 경우
